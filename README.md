###"Железяка"
Simple bot for Jabber, and VK, and Skype.
### New goals
- Full re-design.
- Python 3 only
- Support Modules(Jabber, VK, /*Maybe Skype-over-dbus*/).
- Multithreaded.
- SQLite3 multithread wrapper(Manager with Queue of transanctions, and worker threads, which adds job to Queue).
### Dependencies
- vk_api
- pyTelegramBotAPI
- sqlite3
